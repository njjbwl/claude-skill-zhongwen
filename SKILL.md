# 中文终端 UI 壳

当你看到这个技能被加载，说明用户希望接下来的交互呈现**中文终端界面风格**。

## 核心规则

1. **顶栏**：每次回复的开头，绘制一个模拟的终端顶栏，显示中文信息
2. **底栏**：每次回复的末尾，绘制一个模拟的终端底栏，显示中文提示
3. **命令帮助**：当用户输入 `/帮助` 或 `/helpzh` 时，显示中文版的命令列表
4. **所有回复内容本身保持中文**（你已经在这么做了）

## 顶栏格式

每次回复的第一段内容之前，输出：

```
╭─ 💬 对话: <对话主题> · 🤖 模型: <当前模型> · 📋 模式: <当前模式> ─────────────────╮
│
```

主题优先从 `Conversation` / `conversationName` 环境变量或当前上下文推断，未知则显示「未命名」。
模型从 `settings.json` 的 `model` 字段或当前上下文获取。
模式从当前上下文推断（开发/调试/审查等）。

## 底栏格式

每次回复的最后一段内容之后，输出：

```
╰─ ⌨ /帮助 看命令 · /清屏 清屏 · /模型 切换模型 · Ctrl+C 退出 ──────────────────────╯
```

## 中文命令映射

当用户输入以下中文命令时，按映射处理：

| 用户输入 | 等价行为 |
|---------|---------|
| `/帮助` 或 `/helpzh` | 显示中文命令帮助列表 |
| `/清屏` | 提示用户实际命令是 `/clear` |
| `/模型` | 显示当前模型信息，提示用户实际用 `/model` 切换 |
| `/模式` | 提示用户实际命令是 `/mode` |
| `/撤销` | 提示用户实际命令是 `/undo` |
| `/重试` | 提示用户实际命令是 `/retry` |

## /帮助 命令输出

当用户输入 `/帮助` 或 `/helpzh` 时，输出以下中文帮助信息：

```
╭─ 📖 中文命令帮助 ───────────────────────────────────────────────╮
│                                                               │
│  🎯  AI 命令（对我输入）：                                      │
│  ───────────────────────────────────────────                    │
│  /帮助 或 /helpzh  → 显示本帮助                                 │
│                                                               │
│  🖥️  终端命令（原生 Claude Code 命令）：                         │
│  ───────────────────────────────────────────                    │
│  /clear       清屏                                              │
│  /model      切换 AI 模型                                       │
│  /mode       切换工作模式（开发/审查等）                          │
│  /init       在当前目录初始化项目                                 │
│  /compact    压缩对话历史                                        │
│  /undo       撤销上一步                                          │
│  /retry      重试上一步                                          │
│  /review     审查当前代码变更                                    │
│  /debug      调试模式                                            │
│                                                               │
│  💡 提示：原生命令保持英文输入，上方已标注中文含义。               │
│                                                               │
╰────────────────────────────────────────────────────────────────╯
```

## 注意事项

- **只在回复的文本层面模拟**，不改任何二进制文件
- 如果用户输入 `/sh` 或 `!` 执行真实 shell 命令，正常执行，不额外加壳
- 如果用户明确说「取消中文壳」「恢复英文」，停止本技能的效果
- 顶栏信息尽量精简，避免喧宾夺主
